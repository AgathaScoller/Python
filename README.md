# PYTHON

Repository dedicated to studies, experiments, and implementations in Python.  
The focus is on mastering language fundamentals, data structures, automation, data analysis, and integration with emerging technologies.

---

## STRUCTURE

- **Basics/** – core syntax, logic, and data manipulation  
- **Automation/** – scripts for process and task automation  
- **Data/** – data exploration, cleaning, analysis, and visualization  
- **AI/** – introductory projects in machine learning and artificial intelligence  
- **Utils/** – auxiliary tools and frequently used functions  

---

## OBJECTIVES

- Strengthen technical proficiency in Python  
- Build a foundation for projects in Data Science, AI, and automation  
- Maintain a clear, traceable record of technical evolution  

---

## ENVIRONMENT

- **Language:** Python 3.12+  
- **Core Libraries:** `pandas`, `numpy`, `matplotlib`, `scikit-learn`, `requests`  
- **Recommended Environment:** VS Code or JupyterLab  

---

## CODE STYLE

Code follows principles of readability, modularity, and efficiency.  
Comments are used only when necessary to clarify technical decisions.  
Naming conventions comply with PEP 8 standards.
